# Expense Report Generator

## Description (English)

This project is a simple **Expense Report Generator** built with Python using only basic concepts and standard libraries.

It reads expense data from a CSV file, processes the information, calculates:
- Total expenses
- Total expenses per category

Finally, it generates a text report summarizing all expenses.

This project can be expanded in the future with:
- Data visualization
- Database integration
- Artificial Intelligence analysis
- Web interface

---

## Descrição (Português)

Este projeto é um **Gerador de Relatório de Gastos**, desenvolvido em Python utilizando apenas conceitos básicos e bibliotecas padrão.

Ele lê dados de gastos a partir de um arquivo CSV, processa as informações e calcula:
- Total de gastos
- Total de gastos por categoria

Ao final, o sistema gera um relatório em texto com o resumo dos gastos.

Este projeto pode ser expandido futuramente com:
- Visualização de dados
- Integração com banco de dados
- Análises com Inteligência Artificial
- Interface web
